# 프로그래밍 언어 활용

## 공부 전략

- 이론 공부는 최소화하고 기출 코드 실행 결과를 손으로 따라간다.
- C, Java, Python의 문법 차이보다 변수 변화, 반복문, 조건문, 배열/문자열 처리 흐름을 우선 본다.

## 핵심 키워드

- 변수
- 자료형
- 연산자
- 조건문
- 반복문
- 배열
- 문자열
- 포인터
- 클래스
- 예외 처리

## 이론 정리

- 이 영역은 별도 이론 암기보다 코드 트레이싱이 중요하다.
- 필요한 문법만 오답에서 역으로 보강한다.

## 기출 포인트

- [ ] 증감 연산자와 대입 연산 순서
- [ ] 반복문 종료 조건
- [ ] 배열 인덱스 변화
- [ ] 문자열 길이와 문자 비교
- [ ] 포인터와 주소/값 구분
- [ ] Java 클래스/상속/오버라이딩 결과
- [ ] Python 리스트/슬라이싱/딕셔너리 결과
