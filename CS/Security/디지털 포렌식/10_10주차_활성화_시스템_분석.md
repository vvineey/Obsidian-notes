
# 1. 활성화 시스템 분석

전통적인 디스크 포렌식 : 
저장매체를 분리하거나 이미징해서 비휘발성 데이터를 분석한다.

라이브 포렌식 :
전원이 켜져 있고 운영체제가 동작 중인 시스템을 대상으로 현재 상태를 수집하고 분석하는 것 
 
- 실행 중인 프로세스 
- 현재 네트워크 연결 
- 로그인 사용자 
- 열린 파일 
- 실행 중인 서비스 
- 메모리에 존재하는 악성코드 
- 복호화된 데이터와 암호키
- 클립보드 내용 


# 2. 라이브 포렌식 

실행 중인 시스템에서 데이터를 수집하는 포렌식 방식 

- 휘발성 데이터 확보
- 현재 침입 상태 확인
- 실행 중인 악성코드 탐지
- 공격자의 네트워크 연결 확인
- 로그인 사용자와 세션 확인
- 메모리 안의 증거 확보

현재 상태를 확인할 수 있다는 장점이 있지만, 분석 도구를 실행하는 행위 자체가 시스템 상태를 변경할 수 있음 
따라서 사용한 도구, 명령어, 실행 시각과 결과를 반드시 기록해야 함 


# 3. 휘발성 데이터/ 비휘발성 데이터 

### 휘발성 데이터 
전원이 꺼지거나 시스템이 재부팅되면 사라질 수 있는 데이터 

- RAM 내용 
- 실행 중인 프로세스 
- 현재 네트워크 연결 
- 로그인 세션 
- 열린 파일 
- 실행 중인 서비스 
- 클립보드 
- 명령어 기록 
- 시스템 시간 
- 메모리 안에 암호키 
- 메모리 기반 악성코드 

### 비휘발성 데이터 
전원이 꺼져도 저장매체에 남는 데이터
