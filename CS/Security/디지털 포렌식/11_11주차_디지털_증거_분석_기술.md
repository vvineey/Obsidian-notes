
# 1. 물리적 획득 

저장장치 전체를 가능한 한 비트 단위로 수집하는 방식이다.
파일시스템이 인식하는 파일뿐 아니라 삭제 데이터, 비할당 여역, 슬랙 공간까지 포함한다. 
디스크 이미징, 스마트폰이나 임베디드 장치에서는 CHIP-OFF, JTAG 같은 방식이 사용될 수 있다. 

- 장점 
삭제 데이터 흔적, 비할당 영역, 슬랙 공간, 메타데이터, 원시 데이터 등을 얻을 수 있다. 

- 단점
데이터 양이 매우 많고 수집 시간이 오래 걸리며 장치 손상 위험과 암호화된 데이터는 해석 어려움 


### CHIP-OFF
메모리 칩을 장치에서 물리적으로 분리한 뒤 데이터를 읽는 방식 

- 스마트폰이 심하게 파손된 경우
- 침수로 정상 부팅이 불가능한 경우
- 장치가 고의로 파괴된 경우
- 운영체제나 인터페이스로 접근할 수 없는 경우

일반적인 흐름:

1. 장치에서 메모리 칩의 위치를 확인한다.
2. 열이나 장비를 이용해 칩을 분리한다.
3. 칩을 세척하고 손상 여부를 확인한다.
4. 전용 리더기나 어댑터에 연결한다.
5. 원시 데이터를 추출한다.
6. 데이터 구조와 파일시스템을 분석한다.
