
# 1. 파일 시스템 

저장매체에 파일과 디렉터리를 어떻게 저장하고 관리할지 정한 논리 체계이다. 
사용자는 파일 탐색기에서 폴더와 파일을 확인하지만, 
실제 디스크에는 섹터와 클러스터, 메타데이터, 할당 정보가 복잡하게 저장되어 있다. 


# 2. 물리적 관점과 논리적 관점 

### 물리적 관점 
저장장치를 섹터 같은 실제 저장 단위로 보는 관점
HDD 플래터, SSD 메모리 셸, 섹터, 물리 주소, 디스크 원시 바이트
### 논리적 관점 
운영체제와 사용자가 인식하는 구조를 중심으로 보는 관점 
C/D 드라이브, 폴더, 파일, 파일명, 파일 경로
