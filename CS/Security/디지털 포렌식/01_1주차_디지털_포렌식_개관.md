# 1. 디지털 포렌식

- 디지털 포렌식은 컴퓨터, 스마트폰, 저장매체, 네트워크, 클라우드 등에 남아 있는 디지털 흔적을 
  수집하고 분석하여 사건의 사실관계를 밝히는 기술
- 보이지 않는 디지털 흔적을 눈에 보이는 증거로 바꾸는 과정이 핵심
- 범죄 수사뿐 아니라 사고 원인 분석, 침해사고 대응, 내부 조사, 민사/행정 분쟁에서 사용 


# 2. 디지털 포렌식 조사의 5대 기본 원칙

1. 정당성 : 적법한 절차로 증거를 수집해야 함 
2. 원본성 : 제출 자료과 원본과 동일해야 함
3. 무결성 : **수집 이후** 증거가 변경되지 않았음을 입증해야 함 
4. 신뢰성 : 분석 절차, 도구, 인력, 환경이 믿을 수 있어야 함 
5. 연계 보관성 : 증거 이동, 보관, 분석 이력을 기록해야 함 


# 3. 디지털 포렌식 조사 수행 과정 

1. 사전 준비 
   조사 목적, 범위, 법적 근거, 도구, 역할을 정한다.
   
2. 증거 수집
   디스크 이미지, 메모리, 로그, 네트워크 패킷, 브라우저 기록 등을 확보한다.
   
3. 데이터 보존
   원본과 사본 구분, 접근 통제, 해시값과 보관 이력 기록한다.
   
4. 조사 분석
   파일 추출, 키워드 검색, 타임라인 구성, 삭제 파일 복구, 로그 분석 등을 수행한다.
   
5. 정밀 검토 및 보고 
   분석 결과와 근거를 정리하고 사실과 추론을 구분한다.
   
6. 법적 대응 및 증언 
   법원 제출 보고서, 증언, 무결성 입증 자료를 준비한다.
   
7. 사후 대응 
   재발 방지, 보안 정책 개선, 교육, 절차 보완을 수행한다.
   
   
# 4. 보안 패러다임 

### 경계 보안 

회사 내부와 외부를 나누고 방화벽이나 VPN 같은 장치로 경계를 지키는 방식
한 번 내부로 들어온 사용자를 지나치게 신뢰할 수 있다는 점이 문제 

### 제로 트러스트 

Never Trust, Always Verify

내부 사용자라도 기본적으로 신뢰하지 않고 계속 검증

- 중앙 집중 정책 관리
- 최소 권한 원칙
- 지속적인 인증과 검증
- 사용자, 기기, 위치, 권한, 행위 확인
- 통신 및 저장 데이터 암호화


# 5. 요약

- 디지털 포렌식은 디지털 흔적을 수집, 보존, 분석하여 사건을 밝히는 기술이다.
- 5대 원칙은 정당성, 원본성, 무결성, 신뢰성, 연계 보관성이다.
- 해시값과 가장 직접적으로 연결되는 원칙은 무결성이다.
- 원본성은 원본과 사본의 동일성을 입증하는 개념이다.
- 연계 보관성은 증거의 이동과 보관 이력을 기록하는 개념이다.
- 제로 트러스트는 중앙 집중 정책 관리, 최소 권한, 지속 검증이 핵심이다.
- 개별 관리는 제로 트러스트 특징으로 부적절하다.
