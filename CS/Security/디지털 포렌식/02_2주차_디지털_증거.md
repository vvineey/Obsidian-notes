# 1. 디지털 증거 

디지털 증거는 컴퓨터나 디지털 장치에 저장되거나 네트워크를 통해 전송된 데이터 중 
사건의 사실관계를 밝히는 데 사용할 수 있는 정보

- 문서 파일
- 이메일
- 메신저 대화
- 사진, 영상, 음성
- 웹 브라우저 방문 기록
- 다운로드 기록
- 쿠키, 캐시
- 시스템 로그
- 네트워크 로그
- 파일 생성, 수정, 접근 시간
- USB 연결 기록
- 로그인 기록


# 2. 보관증거와 생성증거

| 구분          | 의미                       | 예시                                     |
| ----------- | ------------------------ | -------------------------------------- |
| 보관증거        | 사람의 의사나 생각이 담긴 내용 중심의 증거 | 문서 본문, 이메일 내용, 메신저 대화, 사진 내용           |
| 생성증거        | 시스템이 자동으로 만든 기록          | 파일명, 파일 크기, 해시값, 타임스탬프, 로그인 기록, 이벤트 로그 |


# 3. 휘발성 증거와 비휘발성 증거 

|구분|의미|예시|
|---|---|---|
|휘발성 증거|전원이 꺼지면 사라지는 증거|RAM, 실행 중인 프로세스, 네트워크 연결, 로그인 세션, 클립보드|
|비휘발성 증거|전원이 꺼져도 남는 증거|문서 파일, 디스크 이미지, 로그, 레지스트리, 브라우저 기록|

# 4. 디지털 증거의 특성 

### 매체 독립성 
디지털 데이터는 저장매체 자체보다 저장 내용이 중요함!!
같은 파일은 USB에서 하드디스크로 복사해도 내용 동일 -> 같은 정보로 판단 
따라서 원본과 사본의 구분이 중요함 -> 해시값을 통해 동일성을 입증 

### 비가시성 / 비가독성 
디지털 증거는 0, 1로 저장 -> 눈으로 바로 해석할 수 X, 운영체제나 프로그램이 해석해서 보여줌 

### 변조와 복제가 쉬움 
쉽게 복사, 수정됨
파일을 열어보는 행위만으로도 접근 시간이 바뀜 
그래서 원본 보호, 해시값, 연계 보관성 기록이 중요함

### 대규모성 
양이 짱 많음

### 초국경성 
클라우드, 해외 서버, 글로벌 플랫폼에 데이터가 저장될 수 있음, 국가 경계를 넘어 존재 가능함 


# 5. 증거능력과 증명력 
|구분|의미|
|---|---|
|증거능력|법정에서 증거로 사용할 수 있는 자격|
|증명력|그 증거가 사건 사실을 얼마나 설득력 있게 뒷받침하는지|
- 증거 능력 : 법정에 올라갈 수 있나?
- 증명력 : 올라간 뒤에 얼마나 믿을만 한가?


# 6. 위법수집증거배제원칙

위법한 절차로 수집된 증거는 원칙적으로 법정에서 사용할 수 없다.

- 영장 없이 수집한 경우
- 영장 범위를 벗어난 분석
- 참여권을 보장하지 않은 압수


# 7. 전문법칙 

법정에서 직접 말한 것이 아니라 다른 사람의 말이나 문서로 전달된 진술성 증거 
보관증거 (사람의 의사나 생각이 담긴 내용 중심의 증거)는 전문법칙 문제가 생길 수 있음 


# 8. 디지털 증거의 증거능력 요건 
| 요건  | 의미                            |
| --- | ----------------------------- |
| 원본성 | 제출된 증거가 원본과 동일해야 함            |
| 무결성 | 수집 이후 변경되지 않았어야 함             |
| 신뢰성 | 분석 절차, 도구, 인력, 환경이 믿을 수 있어야 함 |


# 9. 요약 

- 디지털 증거는 사건 사실을 밝히는 데 사용할 수 있는 디지털 정보다.
- 디지털 증거는 기술적 분석과 법적 신뢰성이 모두 중요하다.
- 보관증거는 사람의 의사나 내용 중심 증거다.
- 생성증거는 시스템이 자동으로 만든 기록이다.
- 타임스탬프는 보관증거가 아니라 생성증거에 가깝다.
- 휘발성 증거는 전원이 꺼지면 사라질 수 있다.
- RAM, 프로세스, 네트워크 연결은 휘발성 증거다.
- 디지털 증거의 특성은 매체 독립성, 비가시성, 변조/복제 용이성, 대규모성이다.
- 가시성은 디지털 증거의 특성으로 부적절하다.
- 증거능력은 법정에서 사용할 수 있는 자격이다.
- 증명력은 증거가 사건을 얼마나 설득력 있게 뒷받침하는지다.
- 증거능력 요건은 원본성, 무결성, 신뢰성이다.
- 위법수집증거는 원칙적으로 사용할 수 없다.
- 전문증거는 수사기관 마음대로 사용할 수 없다.
