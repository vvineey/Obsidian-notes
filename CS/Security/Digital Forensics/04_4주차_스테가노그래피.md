
# 스테가노그래피 

이미지, 음악, 영상, 문서 같은 디지털 콘텐츠 안에 비밀 데이터를 숨기는 기술 
겉으로는 평번한 파일로 보여서 숨겨진 데이터가 있다는 사실 자체를 감춘다.

- 이미지 안에 비밀번호 숨기기
- 음악 파일 안에 텍스트 숨기기
- 문서 파일 안에 압축 파일 숨기기
- 사진 파일 안에 악성코드나 명령어 숨기기

|구분|암호화|스테가노그래피|
|---|---|---|
|목적|내용을 읽지 못하게 함|정보 존재 자체를 숨김|
|겉보기|암호문처럼 수상해 보일 수 있음|일반 파일처럼 보임|
|핵심|기밀성|은닉성|
|예시|문서를 암호문으로 변환|이미지 안에 문서 숨김|

# 구성 요소 
|용어|의미|
|---|---|
|Carrier|숨겨진 데이터를 담는 겉보기 파일|
|Payload|실제로 숨겨지는 데이터|
