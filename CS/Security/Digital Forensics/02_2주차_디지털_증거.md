# 디지털 증거 

디지털 증거는 컴퓨터나 디지털 장치에 저장되거나 네트워크를 통해 전송된 데이터 중 
사건의 사실관계를 밝히는 데 사용할 수 있는 정보

- 문서 파일
- 이메일
- 메신저 대화
- 사진, 영상, 음성
- 웹 브라우저 방문 기록
- 다운로드 기록
- 쿠키, 캐시
- 시스템 로그
- 네트워크 로그
- 파일 생성, 수정, 접근 시간
- USB 연결 기록
- 로그인 기록


# 보관증거와 생성증거

| 구분          | 의미                       | 예시                                     |
| ----------- | ------------------------ | -------------------------------------- |
| 보관증거        | 사람의 의사나 생각이 담긴 내용 중심의 증거 | 문서 본문, 이메일 내용, 메신저 대화, 사진 내용           |
| 생성증거        | 시스템이 자동으로 만든 기록          | 파일명, 파일 크기, 해시값, 타임스탬프, 로그인 기록, 이벤트 로그 |


# 휘발성 증거와 비휘발성 증거 

|구분|의미|예시|
|---|---|---|
|휘발성 증거|전원이 꺼지면 사라지는 증거|RAM, 실행 중인 프로세스, 네트워크 연결, 로그인 세션, 클립보드|
|비휘발성 증거|전원이 꺼져도 남는 증거|문서 파일, 디스크 이미지, 로그, 레지스트리, 브라우저 기록|
