## 시험 범위

- 1-7주차: 디지털 포렌식 개관, 디지털 증거, 저장매체 이미징, 스테가노그래피, 데이터 해석, 컴퓨터 시스템
- 8주차: 중간고사/복습 주차
- 9-13주차: 윈도우 아티팩트, 활성 시스템 분석, 증거 분석 기술, 파일시스템, 복구

## 과제 연결

- 과제 1: 저장매체 이미징, 해시 검증, EXIF, MAC Time, 파일 시그니처
- 과제 2: 스테가노그래피, OpenStego, ZIP 해제, BMP 분석
- 과제 3: DOS Time, Endian, 파일 시그니처, 위장/손상 파일 탐지
- 과제 4: 활성 데이터 비교, MD5 검증, 서비스/포트/계정/의심 파일 분석
- 과제 6: 법적 절차, 연계보관성, 쓰기방지, 디스크 이미징, 법정 검증
