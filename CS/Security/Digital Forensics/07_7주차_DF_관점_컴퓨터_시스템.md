

# 1. 컴퓨터 시스템 계층 

|계층|설명|예시|
|---|---|---|
|사용자|시스템을 사용하는 사람|문서 작성, 웹 검색, 파일 복사|
|애플리케이션|사용자가 실행하는 프로그램|브라우저, 메신저, 문서 편집기|
|운영체제|하드웨어와 프로그램을 관리|Windows, Linux, macOS|
|하드웨어|물리적인 시스템 구성요소|CPU, RAM, SSD, HDD|
사용자가 문서를 열면 애플리케이션 기록이 남고, 운영체제에는 최근 파일/레지스트리/로그가 남고, 저장장치에는 파일 데이터와 메타데이터가 남는다.

# 3. CPU

컴퓨터의 명령어를 해석하고 연산을 수행하는 장치 

- 프로그램 명령 실행 
- 산술 및 논리 연산 
- 데이터 처리 
- 장치 제어 


# 4. 주기억장치와 보조기억장치 

|구분|대표 장치|특징|
|---|---|---|
|주기억장치|RAM|빠르지만 전원이 꺼지면 데이터가 사라짐|
|보조기억장치|SSD, HDD, USB|전원이 꺼져도 데이터가 남음|
RAM은 CPU가 현재 실행 중인 프로그램과 데이터를 빠르게 사용하도록 돕는다.
SSD와 HDD는 운영체제, 프로그램, 사용자 파일, 로그를 장기간 저장한다. 

+ HDD : Hard Disk Drive 
+ SSD  : Solid State Drive 


# 5. 데이터 유지 기준에 따른 분류 

### 휘발성 메모리 
전원이 꺼지면 데이터가 사라지는 메모리 
- RAM 
- CPU 캐시 

### 비휘발성 메모리 
전원이 꺼져도 데이터가 남는 메모리 또는 저장장치 
- SSD 
- HDD
- USB
- SD 카드 
- ROM



# 6. 저장 원리에 따른 분류 

### 반도체 저장매체

- 빠른 속도
- 충격에 강한 편
- 이동식 장치에 많이 사용
- 전자 회로에 데이터 저장

### 자기 저장매체

- 가격 대비 대용량
- 기계식 구조
- 충격에 취약
- 삭제 데이터가 물리적으로 남을 수 있음

### 광학 저장매체

- 레이저로 데이터를 읽고 씀
- 오래된 백업이나 증거물에서 발견될 수 있음
- 현재 사용 빈도는 감소함

- NOR 플래시 
- NAND 플래시
