# 디지털 포렌식

- 디지털 포렌식은 컴퓨터, 스마트폰, 저장매체, 네트워크, 클라우드 등에 남아 있는 디지털 흔적을 
  수집하고 분석하여 사건의 사실관계를 밝히는 기술
- 보이지 않는 디지털 흔적을 눈에 보이는 증거로 바꾸는 과정이 핵심
- 범죄 수사뿐 아니라 사고 원인 분석, 침해사고 대응, 내부 조사, 민사/행정 분쟁에서 사용 


# 디지털 포렌식 조사의 5대 기본 원칙

1. 정당성 : 적법한 절차로 증거를 수집해야 함 
2. 원본성 : 제출 자료과 원본과 동일해야 함
3. 무결성 : 수집 이후 증거가 변경되지 않았음을 입증해야 함 
4. 신뢰성 : 분석 절차, 도구, 인력, 환경이 믿을 수 있어야 함 
5. 연계 보관성 : 증거 이동, 보관, 분석 이력을 기록해야 함 


# 디지털 포렌식 조사 수행 과정 

1. 사전 준비 
   조사 목적, 범위, 법적 근거, 도구, 역할을 정한다.
   
2. 증거 수집
   디스크 이미지, 메모리, 로그, 네트워크 패킷, 브라우저 기록 등을 확보한다.
   
3. 데이터 보존
   원본과 사본 구분, 접근 통제, 해시값과 보관 이력 기록한다.
   
4. 조사 분석
   파일 추출, 키워드 검색, 타임라인 구성, 삭제 파일 복구, 로그 분석 등을 수행한다.
   
5. 정밀 검토 및 보고 
   
   
6. 법적 대응 및 증언 
7. 사후 대응
