# MAC : Mandatory Access Control 
강제적 접근 제어 : 중앙 관리자가 직접 주체와 객체의 권한을 부여하고 접근을 제한한다.

[[10주차 MAC, Bell-LaPadula, Biba, RBAC, ABAC]]

# DAC : Discretionary Access Control
임의적 접근 제어 : 객체의 소유자가 다른 사용자에게 권한을 부여하거나 회수한다.

[[9주차 접근제어 개요, 설계·구축, DAC]]


# TDE : Transparent Data Encription
DBMS가 데이터를 디스크에 저장할 때 자동으로 암호화하고, 메모리로 읽을 때 자동으로 복호화한다. 

[[12주차 데이터베이스 암호화 설계·구축, 키 관리, TDE]]


# HSM : Hardware Security Module
암호키를 안전한 하드웨어 장치 내부에서 생성,저장, 사용된다.
